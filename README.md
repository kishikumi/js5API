# ①課題番号-プロダクト名
js5API_ストレス分析アプリ

## ②課題内容（どんな作品か）
音声できょうのできごとをしゃべるとテキスト化され、
Googleの感情分析APIと連携されてストレス度が表示される。
ストレス度が８０％以上の場合は、誰かの相談するよう、
キャラクターが表示される。
